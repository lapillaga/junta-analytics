About Water Consumption Validation

Our system analyzes water consumption data to identify potential anomalies. Anomalies might indicate:

  • Meter reading errors
  • Water leaks in the distribution system
  • Unusual consumption patterns
  • Potential water quality issues

The system uses two methods to detect anomalies:

Rule-based validation
Checks if the consumption value falls outside the predefined normal range ({{ config.WATER_PARAMETERS.consumption.min }} - {{ config.WATER_PARAMETERS.consumption.max }} m³).
Machine learning model
Uses an Isolation Forest algorithm trained on historical consumption and rainfall data to detect anomalies. This model considers seasonal patterns and context to provide more accurate results.
